Sales Order with Multi-Warehouse Sources

Main features

  • Enables selection of various warehouses for individual products within sales order lines.
  • After confirming the sales order, it generates delivery orders and updates quantity availability based on the selected source warehouse.
  • After confirming the sales order, it automatically generates delivery orders based on the selected warehouse for each line.
